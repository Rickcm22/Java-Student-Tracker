# Java Student Tracker 🎓

Sistema de consola desarrollado en **Java** para la gestión eficiente de registros estudiantiles. Este proyecto implementa una arquitectura básica de manejo de datos mediante objetos.

## ✨ Características
- **Registro:** Permite agregar estudiantes con un ID único autoincremental, nombre y carrera.
- **Consulta:** Listado completo de estudiantes registrados y búsqueda por ID específico.
- **Eliminación:** Borrado de registros del sistema mediante identificadores únicos.

## 🛠️ Tecnologías y Conceptos Aplicados
- **Lenguaje:** Java 17+.
- **Colecciones:** Uso de `ArrayList` para el almacenamiento dinámico de objetos.
- **POO:** Implementación de clases internas (`Static Inner Classes`) para el modelo de datos.
- **Entrada/Salida:** Interacción con el usuario mediante la clase `Scanner`.

## 🚀 Cómo ejecutar
1. Clona el repositorio.
2. Compila el archivo: `javac GestionEstudiantes.java`.
3. Ejecuta el programa: `java GestionEstudiantes`.
