# Java Student Tracker 🎓

Sistema de consola desarrollado en **Java** para la gestión eficiente de registros estudiantiles, creado como parte de mis prácticas en el **Politécnico Grancolombiano**.

## ✨ Características
- **Registro:** Agregar estudiantes con ID automático, nombre y carrera.
- **Consulta:** Listado completo y búsqueda por ID.
- **Eliminación:** Remoción de registros mediante identificadores únicos.

## 🛠️ Tecnologías
- **Lenguaje:** Java 17+
- **Estructuras:** `java.util.ArrayList` y `Scanner`.
